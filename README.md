# Smart Citizen Kit 2.0 Gases Pro Board [![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)]()

### The core firmware (C++)

The technology used:

* Arduino Zero Core
* Platformio

#### Uploading the firmware to the board

After clonning the repository:

1. In folder ./firmware run:

  * `pio run`

2. This step will **take a while**. Once it finished plug the board over USB and run:

  * `pio run -t upload`
